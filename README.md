## 🎮 游戏操作指南

### 💝 爱心连连看
- **目标**: 连接相同的爱心图案来消除它们
- **操作**: 点击两个相同图案，如果路径畅通就会消除
- **提示**: 点击"提示"按钮获得可消除的配对
- **重排**: 点击"重新排列"按钮打乱剩余图案
- **难度**: 简单(8×6) → 中等(10×8) → 困难(12×10)

### 🌟 超级爱心战机 v4.0
- **操作**: 鼠标移动控制飞机，自动射击
- **特色**: 华丽的粒子特效和成就系统
- **目标**: 击败敌机，获得高分，解锁成就

### � 射箭挑战

- **技巧**: 注意风力影响，调整射击角度
- **目标**: 射中目标获得分数

### 🏃‍♀️ 跑酷冒险
- **操作**: 空格键跳跃，S键滑铲
- **目标**: 躲避障碍物，收集道具，跑得越远越好

## 💻 技术亮点

### 🔧 核心技术栈
- **HTML5 + CSS3 + JavaScript**: 纯前端实现，无需安装
- **CSS Grid + Flexbox**: 现代布局技术，响应式设计
- **Canvas API**: 高性能游戏图形渲染
- **Web Audio API**: 动态音效生成
- **LocalStorage**: 本地数据持久化
