# 蓓蓓游戏合集 (Beibei's Games Collection)

这是一个专为蓓蓓设计的有趣小游戏项目集合，包含多种不同类型的互动游戏。

## 🎮 蓓蓓专属游戏系列

### 🏹 蓓蓓的射箭挑战 (Beibei's Archery Challenge)
- 文件：`beibei-archery-challenge.html`
- 说明：物理弹道射箭游戏，考验瞄准技巧和风力判断
- 特色：真实的箭矢轨迹、风力影响、多种目标、成就系统

### 🏃‍♀️ 蓓蓓的跑酷冒险 (Beibei's Parkour Adventure)
- 文件：`beibei-parkour-adventure.html`
- 说明：无尽跑酷游戏，躲避障碍收集道具
- 特色：流畅的跳跃滑铲、多种障碍物、收集品系统、能量机制

### ❤️ 蓓蓓的爱心俄罗斯方块 (Beibei's Heart Tetris)
- 文件：`beibei-heart-tetris.html`
- 说明：爱心主题的俄罗斯方块游戏
- 特色：7种方块类型、爱心特效、等级系统、浪漫配色

## 🕹️ 如何开始游戏

### 快速开始
1. 克隆或下载此仓库到本地
2. 直接在浏览器中打开相应的 HTML 文件即可开始游戏
3. 推荐使用现代浏览器（Chrome、Firefox、Safari、Edge）

### 蓓蓓游戏控制说明

#### 🏹 射箭挑战
- **鼠标**：瞄准和调整射击角度
- **左键点击**：蓄力射箭
- **目标**：射中目标获得分数，注意风力影响

#### 🏃‍♀️ 跑酷冒险
- **键盘控制**：
  - `空格键` 或 `W` - 跳跃
  - `S` 或 `下箭头` - 滑铲
  - `A/D` 或 `左右箭头` - 移动（未来功能）
- **触屏控制**：点击屏幕下方的按钮
- **目标**：避开障碍物，收集道具，跑得越远越好

#### ❤️ 爱心俄罗斯方块
- **键盘控制**：
  - `左右箭头` - 移动方块
  - `下箭头` - 快速下落
  - `上箭头` 或 `空格键` - 旋转方块
- **目标**：消除行数，获得高分

## ✨ 游戏特色

### 🎨 视觉效果
- 精美的渐变背景和动画效果
- 丰富的emoji表情和可爱元素
- 响应式设计，适配不同屏幕尺寸

### 🔊 音效系统
- 动态生成的游戏音效
- 背景音乐和互动音效
- 支持音量调节

### 🏆 成就系统
- 多种成就等待解锁
- 实时统计和进度显示
- 个人最佳记录保存

### 📱 跨平台支持
- 支持桌面浏览器
- 支持移动设备触屏操作
- 自适应高分辨率显示器

## 💻 技术栈

### 前端技术
- **HTML5** - 游戏结构和语义化标记
- **CSS3** - 动画效果、渐变背景、响应式设计
- **JavaScript (ES6+)** - 游戏逻辑、事件处理、动画控制
- **Canvas API** - 游戏图形渲染
- **Web Audio API** - 动态音效生成

### 游戏特性
- **物理引擎** - 重力、碰撞检测、轨迹计算
- **状态管理** - 游戏状态、分数系统、生命值管理
- **本地存储** - 最高分记录、游戏设置保存
- **触控支持** - 移动设备手势操作
- **高分辨率适配** - 支持各种屏幕尺寸和分辨率
